# SQL-DWH_Project
Building a modern data warehouse with SQL Server, including ETL Process ,data modeling and analytics.

Project Overview
This project implements a Data Warehouse solution designed to integrate data from multiple sources and support analytical reporting and business intelligence. The system extracts raw data, transforms it into a clean and structured format, and loads it into a dimensional data warehouse optimized for querying and analysis

🎯 Objectives

Build a centralized data warehouse for analytical queries.

Implement an ETL pipeline to process and clean data.

Design a dimensional model using fact and dimension tables.

Enable efficient reporting and data analysis.
